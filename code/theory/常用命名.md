# 常用命名
* ans: (answer), 答案的意思，用来保存返回的结果。
* retry: 重试
* manage: 管理
* preorder: 先序 
* inorder: 中序 
* rhs: 右值引用
* reuse: 复用


## 类

* Util: 功能、功用


## 函数

### 判断
* valid: 判断是否有效

### 动宾
* uncompressed: 未压缩的
* compress: 压缩
* decompress: 解压
* drain: 排出（代码中用于取出并返回所有的元素，并销毁以前存放元素的容器）、喝干
* take: 取出首部的元素， 一般用于队列中。
* put: 和 take 对应， 即放入元素。
