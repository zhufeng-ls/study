## 变长结构体

**写法**:
```
struct v_struct { 
    int i; 
    int a[0];
};
```

变长结构体需要在堆上使用`malloc`申请空间，因为栈上的空间是函数参数入栈之前就确定好了的。

**变长结构体和指针的区别**:

1. 变长数组只能放在结构体尾部，而指针可以放在任何地方
2. 变长数组不占内存，它只是一个占位符，而指针会占用内存空间。
3. 变长结构体可以直接释放对象，而指针需要先释放自己，在释放结构体。
4. 变长结构体不能放在 c++ 的类中，因为编译器会将一些额外的信息放在类的最后，例如 vptr 或者虚基类。
