# 技巧工具

## 分屏

安装 `Terminator`, ubuntu 默认的仓库就可以使用
```
$ sudo apt-get install terminator
```

## 屏幕录制

安装 recordmydesktop， 默认源

## 便签
安装 indicator-stickynotes

安装 Beyond Compare
